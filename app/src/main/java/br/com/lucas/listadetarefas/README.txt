O projeto se trata de:
 - App feito em kotlin.
 - Utiliza o jetpack compose para criação de componentes gráficos.
 - Utiliza um banco de dados SQLite.

 Sobre o uso:
 - Para o cadastro de uma tarefa deve-se:
    - Escrever o nome da tarefa. (Descrição)
    - Selecionar uma data prazo. (Data de validade)
    - Clicar no botão à direita do campo de texto
 - Para ordenar a lista deve-se:
    - Clicar no botão de lista no canto superior esquerdo da tela
    - Selecionar uma das duas opções de vizualização das tarefas
        - Pode ser ordenado por data de inserção
        - Pode ser ordenada por data de validade (Mais urgente - Menos urgente)
 - Para filtrar as tarefas por palavras chave deve-se
    - Escrever a palavra chave (ou partes de uma palavra ex: Faculdade ou facul)
    - Clicar no botão de pesquisa à direita do campo de texto
 - Para Alterar uma tarefa:
    - Clicar no botão de edição dentro do cartão da tarefa a se editar
    - Mudar o texto da tarefa (Que está exibido no campo de adição de uma nova tarefa)
    - Clicar no botão de salvar alteração, à direita do campo de texto
 - Para deletar uma tarefa:
    - Clicar no botão de lixeira no cartão da tarefa

 Dos desafios adicionais:
 - Implemente a capacidade de editar tarefas existentes. (Feito)
 - Adicione um campo de data de vencimento para cada tarefa. (Feito)
 - Permita que os usuários classifiquem as tarefas por data de vencimento ou por ordem de criação. (Feito)
 - Adicione um mecanismo de pesquisa para filtrar tarefas com base em palavras-chave. (Feito)
